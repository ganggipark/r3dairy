# Markdown 형식 스펙 (MARKDOWN_FORMAT_SPEC.md)

> 사용자에게 노출되는 일간 콘텐츠의 Markdown 형식 기준

## 개요

### 목적

**JSON 스키마 → Markdown 변환**: 구조화된 데이터를 사람이 읽기 쉬운 형식으로 변환

```
DAILY_CONTENT_SCHEMA.json
       ↓
Markdown Generator (역할 번역 적용)
       ↓
{date}.md (웹/앱/PDF 렌더링)
```

### 사용자 대상

- 웹 앱 사용자 (실시간 읽음)
- 모바일 앱 사용자 (다운로드)
- PDF 다이어리 사용자 (인쇄)
- API 클라이언트 (Markdown 파싱)

### 전문 용어 정책

**절대 금지 (사용자 노출 안 됨)**:
- 사주명리, 기문둔갑, 색은식 (전문 용어)
- 천간, 지지, 십성, 오행
- NLP, 알고리즘, 계산 용어

**허용 (사용자 친화적 언어)**:
- 오늘의 흐름, 리듬, 에너지
- 집중, 결정, 관계, 건강
- 방향, 시간대, 활동

## 파일 구조

### 파일 이름

```
backend/daily/{YYYY-MM-DD}.md

예:
backend/daily/2026-01-31.md
backend/daily/2026-02-01.md
```

### 인코딩

- **형식**: UTF-8
- **라인 끝**: LF (`\n`)
- **BOM**: 없음

### 파일 크기

| 기준 | 최소 | 목표 | 최대 |
|------|------|------|------|
| 좌측 페이지 글자 | 400자 | 700-1200자 | - |
| 전체 파일 크기 | 3KB | 5-8KB | - |
| 섹션 개수 | 10 | 15-20 | - |

---

## 헤더 구조

### 파일 시작 (필수)

```markdown
# {YYYY년 M월 D일} | 오늘의 흐름

{오늘의 요약 문단, 1-2문장}

---
```

**예시**:

```markdown
# 2026년 1월 31일 | 오늘의 흐름

오늘은 집중력이 강하고 결정력이 있는 날입니다. 어려운 업무도 수월하게 진행할 수 있는 좋은 에너지입니다.

---
```

### 레벨별 제목

| 레벨 | 용도 | 예시 |
|------|------|------|
| `#` | 페이지 제목 (1개만) | `# 2026년 1월 31일 \| 오늘의 흐름` |
| `##` | 섹션 제목 | `## 오늘의 요약`, `## 집중할 포인트` |
| `###` | 소제목 | `### 해야 할 것 (DO)`, `### 좋은 시간` |
| `####` | 부제목 | 사용하지 말 것 |

## 섹션 구성

### 좌측 페이지 (오늘의 안내) - 필수

**목표**: 사용자가 오늘의 방향을 한눈에 파악할 수 있도록 구성

#### 1. 오늘의 요약 (Summary)

```markdown
## 오늘의 요약

[1-2문장으로 오늘의 핵심 요약]

예시:
오늘은 집중력이 강한 날입니다. 중요한 결정이나 어려운 업무를 진행하기에 좋은 에너지입니다.
```

**요구사항**:
- 최소 30자 이상
- 1-2문장만 (간결)
- 사용자의 행동을 직접 지시하지 말 것

#### 2. 핵심 키워드 (Keywords)

```markdown
## 핵심 키워드

- 키워드1
- 키워드2
- 키워드3

예시:
## 핵심 키워드

- 집중
- 결정
- 관계운
```

**요구사항**:
- 최소 3개, 최대 10개
- 각각 2-4자 (가능하면 짧게)
- 불릿 리스트 (`-` 또는 `*`)
- 한글/이모지 모두 허용

#### 3. 리듬 해설 (Rhythm Description)

```markdown
## 리듬 해설

[설명형 문단, 200자 이상]

이 섹션은 오늘의 흐름이 왜 그런지를 설명하는 교육적 콘텐츠입니다.
사용자가 깊이 있는 이해를 할 수 있도록 구체적인 현상을 설명합니다.

예시:
## 리듬 해설

오늘은 전반적으로 명확한 사고와 결단력이 돋보이는 날입니다. 평소에 미뤄왔던 업무나 결정이 필요한 상황에서는 주저함 없이 진행하는 것이 좋습니다. 반면, 관계 영역에서는 타인의 입장을 더욱 섬세하게 고려해야 합니다.
```

**요구사항**:
- 최소 200자 이상
- 설명형 문단 (카드/리스트 아님)
- 일상 언어 사용
- 구체적인 상황 언급

#### 4. 집중할 포인트 (Focus)

```markdown
## 집중할 포인트

- 포인트1
- 포인트2
- 포인트3

예시:
## 집중할 포인트

- 중요한 회의나 협상에 집중하기
- 결정을 미루지 말고 행동하기
- 수행 중인 프로젝트 마무리
```

**요구사항**:
- 최소 2개 항목
- 불릿 리스트
- 각 30-50자 (간결)
- 긍정적 표현 (해야 할 것)

#### 5. 주의해야 할 점 (Caution)

```markdown
## 주의해야 할 점

- 주의사항1
- 주의사항2

예시:
## 주의해야 할 점

- 충동적인 표현이나 결정 자제하기
- 관계에서 일방적 의견 강요 금지
- 과도한 야근 피하기
```

**요구사항**:
- 최소 1개 항목
- 불릿 리스트
- 피해야 할 행동
- 부정적 표현 (하지 말 것)

#### 6. 오늘의 행동 가이드 (Action Guide)

```markdown
## 오늘의 행동 가이드

### 해야 할 것 (DO)

- 행동1
- 행동2
- 행동3

### 피해야 할 것 (AVOID)

- 금지1
- 금지2

예시:
## 오늘의 행동 가이드

### 해야 할 것 (DO)

- 미뤄온 보고서나 제안 발표
- 팀원과의 솔직한 피드백 대화
- 새로운 프로젝트 시작

### 피해야 할 것 (AVOID)

- 중요한 결정 미루기
- 무리한 약속이나 오버 커밋
- 과도한 야근
```

**요구사항**:
- DO: 최소 2-3개
- AVOID: 최소 1-2개
- 각각 20-50자
- 실행 가능한 행동 명시

#### 7. 시간대별 가이드 (Time & Direction)

```markdown
## 시간대별 가이드

### 좋은 시간대

- 시간1 (~)
- 시간2 (~)

### 피해야 할 시간대

- 시간3 (~)

### 좋은 방향

- 방향1
- 방향2

### 피해야 할 방향

- 방향3
- 방향4

예시:
## 시간대별 가이드

### 좋은 시간대

- 오전 9:00-11:00
- 오후 2:00-4:00

### 피해야 할 시간대

- 저녁 6:00-8:00

### 좋은 방향

- 동쪽
- 북동쪽

### 피해야 할 방향

- 서쪽
- 남서쪽
```

**요구사항**:
- 시간대: 24시간 형식 (HH:MM)
- 방향: 8방위 (동, 남동, 남, 남서, 서, 북서, 북, 북동)
- 각 최소 2개 항목

#### 8. 상태 트리거 (State Trigger)

```markdown
## 상태 트리거

**몸짓**: [위기 상황의 제스처]
**말**: [반응적으로 나올 수 있는 말]
**전환**: [더 나은 반응 방식]

예시:
## 상태 트리거

**몸짓**: 한숨을 쉬며 책상에 몸을 기댈 때
**말**: "이건 내 책임이 아니야"
**전환**: 깊게 숨을 쉬며 "내가 할 수 있는 부분이 뭘까?"라고 물어보기
```

**요구사항**:
- 구체적인 신체 반응 표현
- 자주 나오는 말 또는 표현
- 더 건강한 전환 방식 제시
- 각 50-100자

#### 9. 의미 전환 (Meaning Shift)

```markdown
## 의미 전환

[설명형 문단, 80자 이상]

이 섹션은 "같은 상황을 다르게 보는 방법"을 제시합니다.
부정적 감정을 긍정적 재해석으로 전환합니다.

예시:
## 의미 전환

오늘의 강한 에너지는 "자기 주장"으로 느껴질 수 있지만, 실제로는 "명확한 비전을 전달하는 능력"입니다. 동료의 의견을 무시하는 것이 아니라, 자신의 생각을 자신감 있게 표현할 수 있는 날로 해석하면, 이는 리더십 발휘의 기회가 됩니다.
```

**요구사항**:
- 최소 80자 이상
- 부정 → 긍정 재해석
- 설명형 문단
- 감정적 공감 포함

#### 10. 오늘의 질문 (Rhythm Question)

```markdown
## 오늘의 질문

> [자기 성찰 질문]

예시:
## 오늘의 질문

> 오늘 나의 결정과 판단이 진정 내 마음에서 나온 것일까, 아니면 외부 압력에 의한 것일까?
```

**요구사항**:
- 최소 20자 이상
- 블록 인용 (`>` 사용)
- 자기 성찰 유도
- 일반적인 질문 (개인 상황 X)

## 포맷팅 규칙

### 리스트

```markdown
# 순서 없는 리스트
- 항목 1
- 항목 2
- 항목 3

# 순서 있는 리스트
1. 첫 번째
2. 두 번째
3. 세 번째

# 중첩된 리스트
- 항목 1
  - 부항목 1-1
  - 부항목 1-2
- 항목 2
```

### 강조

```markdown
**굵은 글씨** (중요한 용어)
*이탤릭* (부가 설명)
`인라인 코드` (기술 용어, 매우 제한)
~~취소선~~ (권장하지 않음)
```

**사용 규칙**:
- **굵은글**: 섹션 제목, 중요 개념 (한국어만)
- *이탤릭*: 최소화 (한글 이탤릭은 가독성 낮음)
- 코드: 사용 금지 (사용자 대상)

### 구분선

```markdown
---
```

**사용**: 섹션 구분 (2-3개 최대)

### 블록 인용

```markdown
> 인용 또는 강조 텍스트
```

**사용**: 오늘의 질문, 핵심 메시지

### 테이블

```markdown
| 구분 | 내용 |
|------|------|
| 항목1 | 설명1 |
| 항목2 | 설명2 |
```

**사용**: 최소화 (복잡도 낮춤)

### 이미지

```markdown
![대체 텍스트](이미지_url)
```

**정책**: Markdown 기본만 사용 (HTML 금지)

### 하이퍼링크

```markdown
[링크 텍스트](url)
```

**사용**: 매우 제한 (사용자 분산 주의)

- 🚫 과도한 경고 금지

## 💡 오늘의 요약

명확한 사고와 결단력이 돋보이는 날입니다. 미뤄온 결정들을 진행하기에 좋은 시기입니다.

## 🎯 핵심 키워드

- 집중
- 결정
- 관계운

## 📖 리듬 해설

오늘은 전반적으로 명확한 사고와 결단력이 돋보이는 날입니다. 평소에 미뤄왔던 업무나 결정이 필요한 상황에서는 주저함 없이 진행하는 것이 좋습니다. 반면, 관계 영역에서는 타인의 입장을 더욱 섬세하게 고려해야 합니다. 특히 중요한 대화에서는 자신의 의견을 명확히 하되, 상대방의 피드백도 충분히 수용하는 균형감이 필요합니다.

## 🔥 집중할 포인트

- 중요한 회의나 협상에 집중하기
- 결정을 미루지 말고 행동하기
- 수행 중인 프로젝트 마무리

## ⚠️ 주의해야 할 점

- 충동적인 표현이나 결정 자제하기
- 관계에서 일방적 의견 강요 금지
- 과도한 야근 피하기

## 🚀 오늘의 행동 가이드

### 해야 할 것 (DO)

- 미뤄온 보고서나 제안 발표하기
- 팀원과의 솔직한 피드백 대화 나누기
- 새로운 프로젝트 시작 검토하기

### 피해야 할 것 (AVOID)

- 중요한 결정 미루기
- 무리한 약속이나 오버 커밋
- 과도한 야근 또는 피로 누적

## 🕐 시간대별 가이드

### 좋은 시간대

- 오전 09:00-11:00
- 오후 02:00-04:00

### 피해야 할 시간대

- 저녁 18:00-20:00

### 좋은 방향

- 동쪽
- 북동쪽

### 피해야 할 방향

- 서쪽
- 남서쪽

## 😌 상태 트리거

**몸짓**: 한숨을 쉬며 책상에 몸을 기댈 때
**말**: "이건 내 책임이 아니야"
**전환**: 깊게 숨을 쉬며 "내가 할 수 있는 부분이 뭘까?"라고 물어보기

## ✨ 의미 전환

오늘의 강한 에너지는 "자기 주장"으로 느껴질 수 있지만, 실제로는 "명확한 비전을 전달하는 능력"입니다. 동료의 의견을 무시하는 것이 아니라, 자신의 생각을 자신감 있게 표현할 수 있는 날로 해석하면, 이는 리더십 발휘의 기회가 됩니다.

## 🤔 오늘의 질문

> 오늘 나의 결정과 판단이 진정 내 마음에서 나온 것일까, 아니면 외부 압력에 의한 것일까?

## 💪 건강과 운동

**추천 활동**: 야외 산책, 필라테스, 가벼운 스트레칭

**건강 팁**: 오전 중 물을 충분히 마시기, 저녁 스트레칭으로 긴장 풀기

**웰니스 초점**: 신체의 이완과 마음의 정정

오늘은 활동적인 에너지가 좋으므로 가벼운 운동으로 그 에너지를 긍정적으로 소진하는 것이 좋습니다. 야외 활동은 피로를 모두 풀어주는 효과가 있습니다.

## 🍽️ 식사와 영양

**음식 성향**: 신선하고 상큼한, 자극 없는

**추천 식재료**: 제철 채소, 연어, 아몬드, 블루베리

**피해야 할 음식**: 자극적인 매운 음식, 카페인 과다

오늘은 가벼우면서도 영양 있는 식사가 좋습니다. 아침에 가볍게 시작해서 점심은 든든하게 하는 패턴을 추천합니다.

## 👔 패션과 미용

**의류 스타일**: 깔끔한, 편한, 밝은

**색상 제안**: 파랑, 초록, 흰색

**뷰티 팁**: 자연스러운 메이크업, 보습

오늘의 에너지를 표현하는 색상은 파란색이나 초록색입니다. 이 색상들을 소량이라도 포함하면 하루를 더 긍정적으로 시작할 수 있습니다.

## 💰 쇼핑과 재정

**좋은 구매 아이템**: 필수품, 투자 상품, 자기개발 도서

**재정 조언**: 큰 금액 결정은 신중하게, 충동 구매 제한

**투자 초점**: 장기 계획, 안정성 확보

오늘의 강한 판단력은 재정 결정에 도움이 되지만, 과신하지 않는 것이 중요합니다.

## 렌더링 환경

### 웹 (Next.js)

```tsx
// 마크다운 → HTML 렌더링
import Markdown from 'react-markdown'

export function DailyPage({ markdown }) {
  return <Markdown>{markdown}</Markdown>
}
```

### PDF (WeasyPrint)

```python
# 마크다운 → HTML → PDF
from weasyprint import HTML, CSS

html_content = convert_markdown_to_html(markdown)
HTML(string=html_content).write_pdf('output.pdf')
```

### 모바일 앱

```swift
// 마크다운 파싱 후 네이티브 UI로 렌더링
import Markdown

let parser = MarkdownParser()
let ast = parser.parse(markdown)
let view = MarkdownRenderer().render(ast)
```

# Markdown 파일 검증 체크리스트

- [ ] 파일명: {YYYY-MM-DD}.md

- [ ] 섹션: 최소 10개 이상

- [ ] 링크: 최소화
- [ ] JSON 호환성: 특수문자 이스케이프
- [ ] 역할별: 학생/직장인/프리랜서 표현 확인
