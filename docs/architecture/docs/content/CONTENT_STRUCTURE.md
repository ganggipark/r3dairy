# CONTENT_STRUCTURE - Diary Content Blocks

## 1. Content Pillars
- Rhythm(오늘의 흐름) → Action(실행 가이드) → Record(기록)

## 2. Daily Page (Day)
### Left Page (Guidance) - MUST BE RICH
좌측은 최소 8개 블록, 설명형 문단 포함, 최소 400~600자(권장 700~1200자)
"요약 카드 나열" 금지.

**Standard 10 Blocks (권장)**
1) Today Summary (한 줄 요약)
2) Core Keywords (3~5개)
3) Rhythm Description (2~4문장 해설: 오늘의 기류/흐름을 설명)
4) Focus & Caution (오늘 집중 포인트 / 주의 포인트)
5) Action Guide - Do (3개)
6) Action Guide - Avoid (3개)
7) Time & Direction Guide (좋은 시간/피할 시간/좋은 방향/피할 방향)
8) State Trigger (gesture + phrase)
9) Meaning Shift (의미 전환 문장 1~2문장)
10) Rhythm Question (성찰 질문 1개)

### Right Page (User Writable)
- Schedule / To-do
- Free writing area
- Mood/Energy scores
- Gratitude line
- Tomorrow prep

## 3. Monthly Page (Month)
- Month Theme Summary (2~4문장)
- Top Priorities (3)
- Calendar tiles: 2~3 keyword tags + icons

## 4. Year Page (Year)
- Year Overview (key themes)
- Monthly signals map
- Year Routine Guide (recovery / execute / organize cycles)
