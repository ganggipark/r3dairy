# DAILY_PAGE_LAYOUT - Print & Web Layout

## 1. Daily Page Layout (Print)
### Left Page (Guidance) - Content Rich
- 상단: 날짜/요일
- 본문: 최소 8개 블록
- 문장형 설명(2~4문장) 포함 필수
- 텍스트 밀도: 400~600자 이상 (권장 900자 이상)

### Right Page (Writable)
- 70% 이상 필기 영역 확보
- 구성:
  - 일정/할 일 (체크박스)
  - 자유 기록(라인 또는 점선)
  - 점수(기분/에너지/집중 1~5)
  - 감사 1줄
  - 내일 준비 1줄

## 2. Daily Page Layout (Web)
- Left content as scrollable blocks
- Right as editor area (markdown or plain text)
- Print mode uses same data, different renderer

## 3. Typography Guidance (suggested)
- Left page: readable body text (14~16pt print equivalent)
- Right page: writing-friendly line spacing
