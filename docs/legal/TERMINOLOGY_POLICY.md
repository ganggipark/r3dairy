# TERMINOLOGY_POLICY - User-facing Wording & Legal Safety

## 1. Policy
- 사용자 노출 텍스트는 "일반 언어"만 사용한다.
- 특정 학파/자격/교육 프로그램을 연상시키는 명칭을 피한다.
- 외부 콘텐츠(교재/워크북/강의 문구)를 직접 인용/복제하지 않는다.

## 2. Do Not Use (User-facing)
- NLP, Neuro-Linguistic Programming
- 사주명리, 기문둔갑, 귀문둔갑, 택일, 신살, 십성 등 전문용어
- Practitioner, Certified 등 자격/인증 표현

## 3. Allowed (User-facing)
- 오늘의 흐름 / 리듬 / 기류
- 집중 / 회복 / 정리 / 선택 / 결정 / 관계
- 좋은 시간 / 피할 시간 / 좋은 방향 / 피할 방향
- 상태 전환 / 마음 정렬 / 관점 전환
- 실행 가이드 / 체크리스트 / 기록

## 4. Internal Only
- 계산/모델 문서에서만 사용 가능한 내부 용어는 별도 internal docs로 관리한다.

## 5. Medical Disclaimer (recommended)
- 본 서비스는 의료/치료/진단 목적이 아닌 자기관리 및 기록을 돕는 콘텐츠다.
